Tutorial Download Data Instagram

Panduan lengkap cara mengambil data followers dan following dalam format JSON

Kembali ke Home FAQ

Informasi Penting

Tutorial ini akan membantu Anda mengambil data Instagram dalam format JSON dari pusat akun Instagram. Data ini diperlukan untuk menggunakan Instagram Unfollowers Checker.

1
Akses Pusat Akun Instagram
Langkah pertama adalah mengakses pusat akun Instagram melalui browser web.

Penting

Pusat Akun Instagram:

Buka Pusat Akun Instagram
  • Buka browser web (Chrome, Firefox, Safari, atau Edge)
  • Pastikan Anda sudah login ke akun Instagram
  • Akses link pusat akun Instagram di atas
    • 2
    Request Download Data
    Setelah masuk ke pusat akun, Anda perlu meminta download data Instagram Anda.
    Langkah 2.1: Scroll ke bawah halaman dan cari bagian "Data Download"
    Langkah 2.2: Klik tombol "Request Download" atau "Download Data"
    Langkah 2.3: Masukkan password Instagram Anda untuk konfirmasi
    Langkah 2.4: Klik "Request Download" untuk memulai proses

    Waktu Proses

    Proses download data Instagram membutuhkan waktu 24-48 jam. Instagram akan mengirim email notifikasi ketika data siap untuk didownload.

    3
    Download File ZIP
    Setelah menerima email notifikasi, Anda bisa mendownload file ZIP yang berisi data Instagram Anda.
  • Cek email yang terdaftar di Instagram
  • Buka email dari Instagram dengan subjek "Your Instagram data is ready"
  • Klik link download yang ada di email
  • Masukkan password Instagram untuk mengakses file
  • Download file ZIP ke komputer Anda

    • Struktur File ZIP

    File ZIP akan berisi folder-folder dengan data Instagram Anda, termasuk:

    instagram-data/
├── followers_1.json
├── following.json
├── comments.json
├── likes.json
├── messages.json
└── ... (file lainnya)
    4
    Ekstrak File JSON
    Setelah download selesai, ekstrak file ZIP dan cari file JSON yang diperlukan.

    File yang Diperlukan

    Untuk Instagram Unfollowers Checker, Anda hanya memerlukan 2 file:

    • followers_1.json - Daftar akun yang mengikuti Anda
    • following.json - Daftar akun yang Anda ikuti
  • Extract file ZIP menggunakan WinRAR, 7-Zip, atau extractor bawaan OS
  • Buka folder hasil extract
  • Cari file "followers_1.json" dan "following.json"
  • Copy kedua file ke folder yang sama dengan website
    • 5
    Gunakan Instagram Unfollowers Checker
    Setelah file JSON siap, Anda bisa menggunakan website Instagram Unfollowers Checker.
  • Buka file index.html di browser
  • Upload file followers_1.json dan following.json
  • Klik "Analisis Unfollowers"
  • Lihat hasil analisis dan download jika diperlukan

    • Keamanan Data

    Website ini memproses data di browser Anda sendiri. Data tidak dikirim ke server manapun, sehingga privasi Anda tetap terjaga.

    Pertanyaan yang Sering Diajukan

    Berapa lama waktu yang dibutuhkan untuk download data Instagram?
    Proses download data Instagram membutuhkan waktu 24-48 jam. Instagram akan mengirim email notifikasi ketika data siap untuk didownload.
    Apakah ini aman?
    Yoi men aman karna tidak membutuhkan login pada instagram anda hanya membaca file json yang sudah di download pada instagram anda dan menganalisiis semua isi data followers dan following.
    Apakah data Instagram saya aman?
    Ya, data Anda aman. Website ini memproses data di browser Anda sendiri (client-side). Data tidak dikirim ke server manapun, sehingga privasi Anda tetap terjaga.
    File JSON tidak ditemukan di folder extract, apa yang harus dilakukan?
    Pastikan Anda sudah menunggu 24-48 jam setelah request download. Jika masih tidak ada, coba request ulang download data dari pusat akun Instagram.
    Apakah website ini bisa digunakan untuk akun private?
    Ya, website ini bisa digunakan untuk akun private. Data yang diambil adalah data dari akun Anda sendiri, bukan dari akun lain.
    Apakah ada batasan jumlah followers/following?
    Tidak ada batasan jumlah. Website ini bisa memproses data dengan jumlah followers/following berapapun, namun untuk jumlah yang sangat besar (>10.000) mungkin membutuhkan waktu lebih lama.
